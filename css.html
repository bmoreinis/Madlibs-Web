Interface Designers

Try to get as close as you can to the image below using style.css. For reference, the dashboard element is 500px wide. CSS attributes like size, margin, padding, and text-alignment you already know. New ones are hinted for you, but you'll need to Google W3 schools to get the correct syntax.

Fancy new attributes include border radius (which rounds corners), box shadows, and background linear gradients. Get the simple stuff first.

Whenever possible, combine definitions. For example, the dashboard, textarea and button all have something in common ...

You can't see it, but if you hover over the button, the background turns red and the text turns white. Also, the entire actionbox turns white on hover.
